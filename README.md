# coding-challenge-backend-php

## Starting Project

1. install laravel packs

```composer
composer install
```

1. Run the following command to generate your app key:

```bash
php artisan key:generate
```

2. install npm and run

```npm
npm install
```

```npm
npm run dev
```

## Generate Data

```
php artisan db:seed
```
